Versão corrigida para etiquetas com espaço/gap entre uma linha e outra. O modo padrão agora usa sensor de gap da impressora: a etiqueta tem 25 mm, e o espaço de 4 mm é detectado pela impressora.

1) ZPL do Omie


2) Ajustes de impressão

Move as duas etiquetas para direita/esquerda. Padrão: 55. A etiqueta direita usa X + 320.
Use "Sensor GAP automático" para rolo/folha com espaço entre etiquetas. Esse é o recomendado para corrigir o desalinhamento progressivo.
Use apenas se o sensor GAP não estiver calibrado. Para etiqueta 25 mm em 203 dpi, comece em 200.
Move o conteúdo dentro da etiqueta. Padrão: 10.

3) Revisar produtos e quantidades

Quantidade = número de etiquetas individuais daquele produto. O conversor monta automaticamente as linhas com esquerda + direita.
